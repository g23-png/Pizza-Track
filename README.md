# Pizza-Track

## Objetivo
Desarrollar un sistema de gestión de pedidos para una pizzería utilizando pilas manuales implementadas con listas ligadas en Java.

## Funcionalidades
- Registrar pizzas
- Deshacer pedidos (Undo)
- Rehacer pedidos (Redo)
- Mostrar pedido actual

## Tecnologías utilizadas
- Java
- Visual Studio Code
- GitHub

## Estructura del proyecto
- Pizza.java
- NodoPizza.java
- PilaPizza.java
- GestionPedidos.java
- Main.java
